<h1>RESULTS</h1>

<h3>ArrayList</h3>

<p>get()          = O(1);</p>
<p>set()          = O(1);</p>
<p>add()          = O(N);</p>
<p>add(,)         = O(N);</p>
<p>remove(i)      = O(N);</p>
<p>remove(obj)    = O(N);</p>
<p>add(0,)        = O(N);</p>
<p>add(size-1,0)  = O(N);</p>
<hr>
<h3>LinkedList</h3>

<p>get()          = O(N);</p>
<p>set()          = O(N);</p>
<p>add()          = O(1);</p>
<p>add(,)         = O(N);</p>
<p>remove(i)      = O(N);</p>
<p>remove(obj)    = O(N);</p>
<p>add(0,)        = O(1);</p>
<p>add(size-1,0)  = O(1);</p>
